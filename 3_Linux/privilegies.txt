rwxrwxrwx - права для владельца, права для тех, кто входит в группу владельца, но не является владельцем, права для всех остальных

chmod u-r,g+x,o-r file
chmod u=r,g=w,o=x file

acl - access control list

r = 4
w = 2
x = 1
=> rwxrwxrwx = 4+2+1 4+2+1 4+2+1 7 7 7 

права и иинформация о владельце храняться в специальной структуре inode - индексный дескриптор - на каждый объект файловой системы создается 1 inode 

ls -li - покажет номер inode

информация об имени файла находится в каталоге (специальный тип файла в которой хранятся имена файлов и деректорий)

каталог - соответствие номера inode и имени файла

для каталога x - разрешение на вход в данный каталог
для каталога r - доступ к отображению списка файлов в данном каталоге

после атрибутов доступа у файла/директории идет столбец с количеством имен файла 

ln (link) file_1 file_1_link
